# MindTrack AI

MindTrack AI is a Generative AI-powered mental wellness tracker designed specifically for Indian students preparing for high-stakes exams (JEE, NEET, CUET, CAT, GATE, UPSC).

## 🌟 Features
- **Daily Check-ins**: Log your mood, sleep, study hours, and stress levels.
- **AI Analysis**: Get instant AI-driven insights and coping strategies based on your daily inputs.
- **Progress Tracking**: Visualize your mood and triggers over time with interactive charts.
- **Privacy First**: All data is stored locally on your device.

## 🚀 Live Demo
Access the live web app here: [MindTrack AI](https://socialjignesh.github.io/mindtrack-ai/)

## 🛠️ Technologies Used
- HTML5
- CSS3 (Responsive Grid/Flexbox Design)
- Vanilla JavaScript (Local Storage for state management)
- Chart.js (Data Visualization)

## 💻 Local Setup
Since this is a self-contained web app with no build step:
1. Clone the repository: `git clone https://github.com/socialjignesh/mindtrack-ai.git`
2. Navigate to the directory: `cd mindtrack-ai`
3. Open `index.html` directly in your browser.
